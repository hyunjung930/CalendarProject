# Calendar_Project-Clone_Coding

## 🗓Calendar_Project(클론코딩) - 구글 캘린더 

### 💡선택이유

    평소 구글 드라이브, 구글 미츠를 사용하면서 구글캘린더를 사용해 보고 싶은 생각이 있었고 직접 만들어보면 기능을
    또한 사용해 보고 싶었던 기술 스택이나 툴을(MySQL, Spring, JPA)사용하여 클론코딩을 진행 할 수 있어서 선택하게 되었습니다.
